# 🌸 AI Iris Flower Classifier

A Machine Learning web application that predicts the species of an Iris flower using a trained **K-Nearest Neighbors (KNN)** classification model. The application features a modern, responsive interface built with Flask and provides real-time predictions with confidence scores.

> 🎓 Developed as **Project 2** during my Artificial Intelligence Internship at **DecodeLabs**.

## 📌 Features

- 🌸 Predicts the Iris flower species
- 🤖 Machine Learning model using K-Nearest Neighbors (KNN)
- 🎯 Displays prediction confidence
- 💎 Modern Glassmorphism UI
- 📱 Fully responsive design
- ⚡ Fast predictions using Flask
- 🌍 Deployed on Render

---

## 🧠 Machine Learning Workflow

1. Load the Iris dataset
2. Split the dataset into training and testing sets
3. Train a K-Nearest Neighbors (KNN) classifier
4. Save the trained model using Joblib
5. Load the model into a Flask application
6. Accept user input through the web interface
7. Predict the flower species
8. Display the prediction with confidence
